### $nextTick()
作用：vue中的$nextTick主要用于处理数据动态变化后，DON还未及时更新的问题，用nextTick就可以获取数据更新后最新DON的变化。
场景一：点击获取元素的宽度
场景二：点击按钮原本以v-show=false隐藏起来的输入框，并获取焦点
场景三：增加和修改
